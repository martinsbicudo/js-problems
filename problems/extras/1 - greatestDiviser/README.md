# Description

Find the greatest common diviser of two numbers

## Example

- For `a = 5` and `b = 15`, the output should be `solution(a, b) = 5`.

- For `a = 30` and `b = 60`, the output should be `solution(a, b) = 10`.

- [input] integer a

  Guaranteed constraints: `1 ≤ a.length ≤ Infinity`.

- [input] integer b

  Guaranteed constraints: `1 ≤ a.length ≤ Infinity`.

- [output] integer
